Iringa – Makambako – Lindi – Gaza – Dodoma

Network Overview

This enterprise network is designed using Cisco routers and switches. The system supports automatic IP assignment using DHCP and ensures communication between all branches through WAN connections.

IP Addressing Scheme

Location Network Subnet Mask Gateway
Gangilonga 192.168.11.0 255.255.255.0 192.168.11.1
Makambako 192.168.20.0 255.255.255.0 192.168.20.1
Lindi 192.168.30.0 255.255.255.0 192.168.30.1

Router Configuration Summary

  • Router R1 (Iringa) – Connects Gangilonga and Makambako LANs
  • Router R2 (DSM) – Connects Lindi and Gaza LANs
  • Routers communicate via a point-to-point WAN link

DHCP Services

Dynamic Host Configuration Protocol (DHCP) is used to automatically assign IP addresses to all end devices. This reduces manual configuration errors and improves network scalability.

End Devices

All PCs, laptops, and smartphones obtain IP addresses automatically. Wireless devices connect through a cellular tower and access the same network resources.
